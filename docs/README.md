## 用法

### 直接看 (合成数据)
双击 `index.html`,浏览器打开就能用。

### 用真实数据 (推荐)
```bash
pip install yfinance pandas
python docs/fetch_data.py
```

生成后页面会从 `docs/index.html` 同目录读取 `docs/data.json`。数据源是 Yahoo Finance/yfinance，属于延迟历史行情，不是交易所实时同步盘口。
