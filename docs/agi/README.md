## 用法

### 直接看 (合成数据)

双击 `ai_global_index.html`,浏览器打开就能用。

### 用真实数据 (推荐)

```bash
pip install yfinance pandas
python fetch_data.py
```

会在同目录生成 `data.json`,HTML 自动加载。

## 功能

- 日 / 周 / 月周期切换 + K 线 / 折线切换
- USD / CNY / Local 三币种交叉折算
- 周 / 月再平衡 (锁定权重不漂移)
- 14 条 AI 大事记 hover 浮动介绍
- T₀ 视觉特化标识 (ChatGPT 发布日)
- 成分股 AI 子赛道概念标注
- 50/50 中美策略可视化分割
